# MoodLens

AI-powered emotion detection system using facial expressions and audio analysis.

## Features

- Facial emotion recognition
- Real-time webcam detection
- Audio mood analysis
- Emotion dashboard
- FastAPI backend
- React frontend

## Tech Stack

- Python
- PyTorch
- OpenCV
- FastAPI
- React
- FER-2013 Dataset
